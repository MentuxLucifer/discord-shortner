# Discord URL Shortner

I Just Made URL Shortner Discord, If Use Code Credit Me

## Installation

Just Type

```bash
npm install
```
And Then When All Packages Is Installed Just Type

```bash
node index.js
```

## Usage

``!shorten`` | Shorten URL
